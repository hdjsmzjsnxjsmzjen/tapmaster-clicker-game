## Описание приложения

TapMaster — это мобильная кликер-игра, разрабатываемая с использованием фреймворка Flutter. Приложение предназначено для развлечения пользователей и позволяет зарабатывать игровые очки с помощью нажатий на экран.

Основная проблема, которую решает приложение, — отсутствие простых и доступных игр для коротких игровых сессий. Многие современные игры требуют долгого обучения и значительного времени, в то время как TapMaster предлагает понятную механику и быстрый игровой процесс.

Целевая аудитория приложения — школьники, студенты и пользователи любого возраста, которые хотят отвлечься, расслабиться или провести свободное время за простой игрой. Приложение не требует специальных навыков и подходит для казуального гейминга.
